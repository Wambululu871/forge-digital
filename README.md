##Forge Digital Solutions

High-performance frontend architecture and core web systems for my agency; Forge Digital Solutions.

## Key Features

- Semantic HTML5 structure for accessibility and search engine clarity
- Modular CSS layouts built for maintainability and scalability
- Asset optimization pipeline for minimal load times and improved Core Web Vitals
- Fluid responsiveness across all viewport sizes and device types

## Tech Stack

| Layer      | Technology          |
|------------|---------------------|
| Markup     | HTML5               |
| Styling    | CSS3                |
| Logic      | JavaScript (ES6+)   |
| Deployment | Vercel              |

## The Process

Development centered on asset pipeline efficiency, ensuring images, fonts, and scripts were optimized before reaching production. Layout precision was treated as a hard requirement — not an afterthought — with each component built to hold its structure across breakpoints. Design fidelity was maintained throughout by working directly from source files and validating output against production-grade conditions.

## How to Run

```bash
# Clone the repository
git clone https://github.com/Benwam/forge-digital.git

# Navigate into the project directory
cd forge-digital

# Open the entry point in your browser
open index.html        # macOS
xdg-open index.html    # Linux
start index.html       # Windows
\```
```
